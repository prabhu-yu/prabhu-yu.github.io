  6. numbertext-libre-office
    Often, I want to convert the Number in LibreCalc into Words. Ex: if a cell contains 45 as a number, I want it to be shown as "Fourty five" in another cell. This typically needed when you want to write the Currency in Words. For this, simply use some thing as below.
    ="Amount (in words) Rupees " &NUMBERTEXT(G20)&" only"
    It automatically get converted to ="Amount (in words) Rupees " &org.numbertext.numbertext(G21)&" only"

    https://extensions.libreoffice.org/en/extensions/show/numbertext-1

  7. What is a Static WebPage/Website?
    Generally, Static webPage are compared with Dynamic WebPage/Website.
    All static websites can be as dynamic as any Dynamic website, but it uses, the client site Javascript. Before we understand, what it is, let us see why we need it.
    It is almost functional equivalent of Dynamic web page, and comes with these too.
    - Extra ordinarily Simple
    - implicitly Secure
    - Highly economical (almost you need no money to setup)
    - Fast
    - Nearly resistant to attacks like DoS or DDoS
    - As dynamic as any other Dynamic webpage or website.
    - You never need a expert to setup and manage it.
    If all the pages are served statically, such webpage is called webpage. If all pages served by a website are static webpages, then whole website may be called static website
    How this matter?
    We can write all files for our website once and serve them to users as they are.
    We still need a server, but its job is just to hand over the files, as they are.
    thus, server job is similar to that of file hosting server or CDN.
    Oh, I need dynamic content in my website ...
    Yes, that is totally possible.
    How? we can use JAvascript at client side to provide dynamic feature.
    It is a very old concept but getting mainstream attention.
    Static websites are offer same functionality as that of Dynamic websites, yet, they (=static website) are far simpler, secure, safe, easy to manage, does not require very sophisticated knowledge to develop and maintain. Very first question that comes to mind is , how come static website can offer the dynamic nature of a web page. Answer is simple and well known : everything is handled at client side using JavaScript. The client script will directly talks to amy needed server instead of a dedicated webserver. Infact, in the static website, the web-server will serve the static pages only. Nothing else. No code runs on it. It is like file server. that is all. Any database, will run at a server. the client java script will communicate to it (certainly, by autheticating) Then does what ever it wants. If see properly, there is no fixed web technologies you need to follow as in Flask, Django etc. It is upto developer to do what is best.
    Reference: https://www.internetingishard.com/html-and-css
    https://www.smashingmagazine.com/2015/11/modern-static-website-generators-next-big-thing/
